Nombre del proyecto

SNOOP

Descripcion general

El proyecto consiste en el desarrollo de una plataforma web inspirada en el modelo de Pinterest, orientada a la visualización, organización y descubrimiento de contenido visual. La página permitirá a los usuarios explorar imágenes y publicaciones, compartir ideas de forma sencilla e intuitiva.

La plataforma estará diseñada con una interfaz atractiva y responsiva, priorizando la experiencia de usuario y la navegación fluida. Además, incorporará funcionalidades como registro e inicio de sesión, interacción con publicaciones (dar “me gusta”) y un sistema de búsqueda que facilite el descubrimiento de contenido relevante.

Objetivos

Desarrollar una aplicación web interactiva que permita a los usuarios descubrir contenido visual de manera eficiente, fomentando la creatividad y el intercambio de ideas y momentos.

Diseñar una interfaz visual atractiva, intuitiva y adaptable a distintos dispositivos.

Implementar un sistema de gestión de usuarios con registro y autenticación.

Fomentar la interacción entre usuarios mediante acciones como reaccionar a los post de tu preferencia.

Garantizar un rendimiento óptimo y una experiencia de navegación fluida.
