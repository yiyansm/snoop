# Pinterest-like PHP App (Simple)

Estructura del proyecto y archivos principales. Usa PHP (procedural), MySQL, CSS y JS (AJAX) para la funcionalidad de "like".
- Coloca este proyecto en la raíz de tu servidor (ej. `/var/www/html/pinterest-clone`).
- Crea la base de datos con `db.sql`.
- Asegúrate de que `uploads/` tenga permisos de escritura por PHP.
